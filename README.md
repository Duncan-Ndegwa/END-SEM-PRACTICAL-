# cdflang_677200

## Description

A functional-first domain-specific language prototype for climate sensor monitoring based on CDF-3-2-0-0-1-0-2.

## Features

* Tokenization (front-end processing)
* Symbol table with lexical scope
* Execution engine (simulation)
* Functional constructs (map, abstraction)
* Result-based error handling (Ok / Err concept)
* Robust error detection
